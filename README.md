# egzamin
# 🎓 Panel Egzaminacyjny - Technik Mechatronik

Aplikacja wspomagająca naukę do egzaminów zawodowych (kwalifikacja ELM.03) na podstawie bazy pytań podzielonej na szczegółowe działy tematyczne.

## 🚀 Funkcje
- 👁️ **Szybka Ściąga:** Domyślny tryb podglądu wszystkich pytań wraz z od razu oznaczonymi poprawnymi odpowiedziami.
- ✍️ **Tryb Egzaminu:** Interaktywny quiz umożliwiający losowanie pytań z wybranego działu i zliczanie statystyk (poprawne/błędy).
- 🌙 **Dark Mode:** Domyślny, nowoczesny, ciemny motyw graficzny optymalny do długiej nauki w nocy.
- 📁 **Podział tematyczny:** Oficjalne działy w pełni posegregowane według struktury egzaminacyjnej.
